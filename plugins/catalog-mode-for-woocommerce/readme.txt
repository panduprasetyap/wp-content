Catalog mode for WooCommerce is an addon for WooCommerce which quickly converts your store into catalog mode. It easily removes the add to cart, cart functionality, checkout from your store.

== Installation ==
* Login to your WordPress dashboard and navigate to Plugins > Add New
* Search for Catalog Mode For WooCommerce
* Click install.
* Click Activate.
* Once the plugin is installed then you can see Catalog Mode menu in the left side screen.
* Just go through the settings and save the settings and you are done
